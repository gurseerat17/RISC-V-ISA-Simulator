# RISC-V-ISA-Simulator

Functional Simulator for subset of RISC-V instruction Set
Directory Structure:
  |
  |- SOURCE CODES
      |
      |- PHASE1.cpp
      |- PHASE2.cpp
      |- opcode.txt
      |- data.txt
      |- operation.txt
  |- TESTS
      |
      |- bubble_sort.asm
      |- factorial.asm
      |- fibonacci.asm
  |- OUTPUTS
      |- mcode(sample).mc
      |- data_mem(sample).mc
      |- data_r(sample).mc
      |- output_bubbleSort
      |- output_fibonacci(5)
      |- output_factorial(7!)
  |- DOCS
      |- Design docs
      |- GROUP_README

How to execute
$ g++ PHASE1.cpp
$ ./a.out
$ g++ PHASE2.cpp
$ ./a.out
